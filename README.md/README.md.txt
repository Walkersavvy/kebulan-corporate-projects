## 🧠 Repository Overview

This repository centralizes internal processes, onboarding, and agile collaboration assets across Kebulan Grid’s corporate pillars:

- ⚖️ Legal – Service agreements, compliance templates, IP records  
- 🏗️ Operations – Onboarding flows, ECS checklists, induction guides  
- 🎨 Branding – Guidelines, asset documentation, identity systems  
- 🧭 Project Manager Tools – Sprint boards, retrospectives, contribution protocols

Built for version control, contributor sync, and portfolio visibility in creative-tech environments.
